# SmartGeyserSim
A simple Java simulation of a geyser heating system with selectable modes, room temperature input, and safety checks for overheating and maximum heating time.
# Geyser Heating Simulation

This project simulates a geyser heating system with selectable modes, 
room temperature input, and safety shutdown checks. 

It contains two versions:
- **Non-OOP (Procedural):** Straightforward implementation using loops and conditionals.
- **OOP (Object-Oriented):** Refactored design using classes, encapsulation, and cleaner structure.

Both versions demonstrate how the same logic can be implemented in different programming paradigms.
